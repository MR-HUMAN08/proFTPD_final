ProFTPD 1.3.5 mod_copy RCE Lab

Vulnerability Description

ProFTPD 1.3.5 contains a critical vulnerability in its mod_copy module that allows unauthenticated attackers to execute arbitrary commands on the target system. The vulnerability exploits the SITE CPFR and SITE CPTO commands to copy files from any readable location to a web-accessible directory.

Target Service: ProFTPD 1.3.5
Vulnerable Module: mod_copy
CVE: CVE-2015-3306
Difficulty: Medium
